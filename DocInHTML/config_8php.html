Detailed Description

Цей файл призначен для добавлення прав адміністратора, а також відповідальних функцій за перевірку цілісності бази даних, при відсутності таблиці створює нову із полями, якщо пошкоджена, то тільки повідомляє яка таблиця пошкоджена.
Примір використання файлу
php config.php (addAdmin, deleteAdmin) login (Примір 1)

Parameters
addAdmin- Мітка для виконання надавання доповнених прав.
deleteAdmin- Мітка для виконання пониження доповнених прав.
login- Логін користувача якому надати або позбавити доповнених прав.
php testTable - (примір 2)
testTableПеревірити цілісність таблиць або створити таблицю.
Warning
Потенційно небезпечно зберігати цей файл в публічних папках сервера.
